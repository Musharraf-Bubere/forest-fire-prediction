# 🔥 Forest Fire Prediction Web App

An End-to-End Machine Learning Project that predicts the Fire Weather Index (FWI) using meteorological data. Built with Python, Flask, and Machine Learning.

--------------------------------------------------

📌 PROJECT OVERVIEW

This project predicts the Fire Weather Index (FWI) using input features like temperature, humidity, wind speed, rain, etc. The trained ML model is deployed using a Flask web application where users can input data and get predictions.

--------------------------------------------------

🚀 FEATURES

- End-to-End ML Pipeline  
- Flask Web Application  
- Real-time Prediction  
- Clean Project Structure  
- Model Persistence using Pickle  

--------------------------------------------------

🧠 MACHINE LEARNING WORKFLOW

1. Data Collection

⚙️ INSTALLATION & SETUP

1. Clone Repository

git clone https://github.com/Musharraf-Bubere/forest-fire-prediction.git
cd forest-fire-prediction

2. Create Virtual Environment

python -m venv reg_env

3. Activate Environment

Windows:
reg_env\Scripts\activate

Mac/Linux:
source reg_env/bin/activate

4. Install Dependencies

pip install -r requirements.txt

5. Run Application

python application.py

6. Open Browser

http://127.0.0.1:5000/
